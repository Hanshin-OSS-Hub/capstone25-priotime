## 2025-01-09
- 기초 구조 업로드 (React - node.js(express) - mongoDB Atlas
- 사용 방법 :
  1. <> Code 버튼(초록색) 클릭 -> Open with GitHub Desktop 클릭하여 프로젝트를 로컬 저장소에 받아온다.
  2. 보안을 위해 .env 파일(환경변수)은 다른 경로를 통해 공유될 예정.
  3. .env 파일은 server 폴더 안에 넣을 것. 또한 IP 주소 허용을 위해 팀장에게 문의할 것. (mongoDB Atlas 접근 권한이 필요함. 되도록 한 아이피로만 접근하는 것 권장함)
  4.  경량화를 위해 업로드된 파일이라 각자 PC에서 설치를 권장하는 파일이 있음.
  5.  터미널을 2개 만든다.
  6.  하나는 cd server 를 입력하고 npm install을 입력 (서버용 터미널)
  7.  다른 하나는 cd client 를 입력하고 npm install을 입력 (클라이언트용 터미널)
  8.  서버용 터미널에 node server.js 입력하여 실행. 실행 취소는 ctrl + c 입력하면 실행이 취소되며 터미널 입력이 다시 활성화됨.
  9.  클라이언트용 터미널에 npm start를 입력 후 기다리면 크롬 창이 생성되며 웹 페이지 결과물을 볼 수 있음.
  10.  테스트 해보고 어렵다면 팀장에게 문의할 것.
